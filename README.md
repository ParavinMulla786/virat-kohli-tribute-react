# 🌟 Virat Kohli Tribute React App

A React-based tribute web application dedicated to **Virat Kohli**, built to practice and strengthen core React concepts such as components, props, reusable UI structure, and basic state management.

---

## 🚀 Features

- 🧩 Component-based architecture
- 📦 Data passing using props
- 🎨 Clean and responsive UI design
- 📱 Mobile-friendly layout
- ⚛️ React fundamentals practice project

---

## 🛠️ Tech Stack

- React JS
- JavaScript (ES6+)
- HTML5
- CSS3

---
